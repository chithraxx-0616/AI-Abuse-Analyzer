# Social Media Abuse Prevention & Moderation Platform

## 📄 Description
A browser extension + AI-powered moderation platform that detects abusive comments and overlays warnings directly on social media feeds. Built with React, Node.js, and Hugging Face NLP models for Next Gen Cyber Defense.

## 🏗️ Tech Stack
- **Frontend/Extension:** React.js, Tailwind CSS, JavaScript, Manifest v3
- **Backend:** Node.js + Express, MongoDB
- **AI/NLP:** Hugging Face Transformers (BERT/RoBERTa), spaCy/NLTK
- **Version Control:** GitHub

## 👩‍💻 Team Roles
- Member 1: Browser Extension + Frontend
- Member 2: Backend + APIs
- Member 3: AI/NLP Engine

## 🚀 Demo Plan
1. User opens Instagram
2. Extension scans comments/posts.
3. Abusive content flagged by AI engine.
4. Overlay highlights abusive text in feed.
5. Dashboard shows flagged content + analytics.

## ⚡ Setup Instructions
1. Clone repo: `git clone https://github.com/your-team/social-moderation-project.git`
2. Navigate to folder: `cd social-moderation-project`
3. Run backend: `cd backend && npm install && node server.js`
4. Run AI engine: `cd ai-engine && pip install -r requirements.txt && python app.py`
5. Load extension: Open Chrome → Extensions → Load unpacked → select `extension/`
